# Aapanel

Aapanel docker compose file to customize

## Dependencies

- Docker
- Docker compose

## Quick Installation

```
git clone https://github.com/antonio24073/aapanel.git
cd aapanel
```

or download and extract zip

## Instructions

To run do this command in the project root folder:

```
docker compose up -d --build 
```

To stop:

```
docker compose down
```
